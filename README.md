# MoneyFlow — твой личный финансовый центр

Умный список покупок с отслеживанием цен  
Менеджер подписок и регулярных платежей  
Полный учёт автомобиля (заправи, ТО, расходы)  
Единая красивая аналитика и прогнозы  

Технологии: Python + Flet (flat design) + PostgreSQL + SQLAlchemy

Лицензия: Apache 2.0
